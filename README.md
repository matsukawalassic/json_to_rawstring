# json_to_rawstring

`{"example": ".com"}`

goes

`"{  \"example\": \".com\"}"`

# Preconditions

## Infras

node - over 8.9.3

## Installation

```
cd /path/to/json_to_rawstring
npm install
```

## prepare json

ex.)

/path/to/json_to_rawstring/example.json

```
{"example": ".com"}
```

## run

```
cd /path/to/json_to_rawstring
node index.js example.json
```

## see

`output/example.json`
